## 使用方法

 1. 先安装**nonebot_plugin_buff**
 2. 启用此插件，具体请看**nonebot**官网文档
 3. 在入口文件下面的**search_data**目录下添加最新的**data.txt**
 4. 发送 **查询/查 xxx** ，然后跟随bot指令即可查询价格

**！！！！！一定要下载数据集data.txt并放入search_data才可以使用！！！！！**
